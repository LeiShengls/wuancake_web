二.HTML的几个定义

1.HTML

  • Hyper Text Markup Li>
  • 使用标记标签来描述网页

2.HTML标签

  • HTML标记标签通常被称为 HTML标签 (HTML tag)。
  • 是由尖括号包围的关键词，比如 <html>
  • HTML标签通常是成对出现的，比如 <b> 和 </b>
  • 前一个为开始标签，后一个为结束标签。也被称为开放标签和闭合标签

3.HTML 文档 = 网页

  • HTML 文档描述网页
  • HTML 文档包含 HTML 标签和纯文本
  • HTML 文档也被称为网页
